## Datasets

Para este codigo se requiere bajar los siguientes datasets disponibles públicamente.

Dataset PyShoe: https://github.com/utiasSTARS/pyshoe

Dataset HelmetPoser: https://lqiutong.github.io/HelmetPoser.github.io/

## Estructura de Carpetas

Clonar el repositorio dentro de la carpeta <<Código>> siguiendo la siguiente estructura de carpetas.
En el fichero Principal.py modificar la ruta de la carpeta raíz para acceder a los datasets.

<!-- TREEVIEW START -->
```bash
├── root/
│   ├── Codigo/
│   ├── Datasets/
│   │   ├── DatasetPyShoe/
│   │   └── DatasetHelmetPoser/
```
